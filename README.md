### 📱LAB2/ LocalTaxCalculator

# Description

LocalTaxCalculator est une application Android développée avec Android Studio permettant de calculer le montant total des impôts locaux en fonction de :

- La surface du logement (en m²)
- Le nombre de pièces
- La présence d’une piscine


# Objectif

Ce projet a été réalisé dans le cadre du LAB2 Android pour apprendre :

-La création d’interfaces graphiques (XML)
-La gestion des entrées utilisateur (EditText, CheckBox)
-Le traitement logique en Java
-L’affichage dynamique des résultats
--> Formule de calcul
  - Impôt de base = surface × 2
  - Supplément = (nombre de pièces × 50) + (100 si piscine)
  - Total = impôt de base + supplément
